При переключении между экраном "Избранное" и "Поиск", если были показаны все вакансии - видимыми становятся только первые 3 и происходит прокрутка к элементу с index==0, как при изначальном заходе на экран.

Если же открыт поиск - экран останется в состоянии поиска, поскольку выход из поиска предусмотрен по ТЗ, а выход из просмотра всех вакансий нет.

Также в приложении запрос в сеть был реализован таким образом, чтобы при повторном запуске приложения данные не перезаписывались для сохранения состояния добавленных вручную в избранное элементов.
